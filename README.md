# 📊 Task 3 - Data Visualization

## Project Description

This project demonstrates how raw data can be transformed into meaningful visualizations using Python. The project uses Pandas for data handling, Matplotlib and Seaborn for creating charts, and presents insights that support data-driven decision-making.

## Objectives

* Transform raw data into visual formats.
* Create charts and graphs using Matplotlib and Seaborn.
* Analyze book data through effective visualizations.
* Present meaningful insights using data storytelling.
* Build a portfolio-ready data visualization project.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* NumPy

## Dataset

The project uses a book dataset containing the following attributes:

* Book Name
* Author
* Genre
* Rating
* Price

## Visualizations

* Books by Genre (Bar Chart)
* Top 10 Authors (Bar Chart)
* Rating Distribution (Histogram)
* Price Distribution (Histogram)
* Price Box Plot
* Price vs Rating (Scatter Plot)
* Correlation Heatmap

## Key Insights

* Identified the most common book genres.
* Identified the authors with the highest number of books.
* Analyzed the distribution of book prices and ratings.
* Compared book prices with ratings.
* Examined correlations between numerical variables.
* Generated visual insights to support decision-making.
